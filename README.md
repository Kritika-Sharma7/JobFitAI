## Overview
JobFitAI helps job seekers analyze how well their resume matches a given job description.  
It provides a **job fit score**, **skills gap analysis**, and **actionable suggestions** to improve resumes for better ATS compatibility.

This project is being built incrementally with a strong focus on **clean APIs, scalable architecture, and real-world use cases**.
